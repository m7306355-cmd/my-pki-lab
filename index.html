💡 如何验证证书链

点击浏览器地址栏左侧的锁头图标（或"不安全"提示），选择查看证书详情， 即可看到完整的三层证书链：My Root CA → My Intermediate CA → localhost。 由于根证书未加入浏览器信任列表，浏览器会提示证书不受信任，这是自建 PKI 的正常现象。

🛠 技术栈

本站使用 OpenSSL 生成 RSA 密钥对和 X.509 证书， 使用 Python 内置的 ssl 模块和 http.server 启动 HTTPS 服务器。 证书编码格式为 PEM (Base64 + DER)， 私钥格式为 PKCS#1， 证书签名请求格式为 PKCS#10 (CSR)。
